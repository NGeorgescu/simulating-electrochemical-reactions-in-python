# Simulating Electrochemical Reactions in Python

Welcome. This is a Python-native adaptation of Michael Honeychurch's
*Simulating Electrochemical Reactions in Mathematica* (SERM): a complete,
runnable course in the digital simulation of electrochemical reactions, built
on NumPy, SciPy, matplotlib, and SymPy. Every chapter is a Jupyter notebook,
and each is validated against the closed-form results of electroanalytical
chemistry (Cottrell, Randles–Sevcik, Nicholson–Shain, Sand, Levich,
Butler–Volmer kinetics, and a Monte Carlo random-walk model), so the simulation
code is verified from first principles rather than merely plausible.

Source code, notebooks, and issue tracker live on GitHub:
[NGeorgescu/simulating-electrochemical-reactions-in-python](https://github.com/NGeorgescu/simulating-electrochemical-reactions-in-python).

All physics, algorithms, and the structure of the worked examples are due to
Michael Honeychurch; this project reimplements them as idiomatic Python.
